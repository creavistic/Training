HOMBRO TRAINER - APP PARA ANDROID

Qué incluye:
- App web instalable para Android.
- Rutina editable: ejercicios, series, repeticiones, segundos por serie y descansos.
- Avance automático ejercicio por ejercicio.
- Voz y pitidos opcionales.
- Imágenes de los 4 ejercicios del hombro.
- Guarda la configuración en el móvil.
- Funciona sin internet después de abrirla una vez.

Cómo usar en Android:
1. Descomprime el ZIP.
2. Sube la carpeta a cualquier hosting sencillo o ábrela desde un servidor local.
   Opción fácil: enviarla a un hosting tipo Netlify, GitHub Pages, IONOS, etc.
3. En Android abre la URL con Chrome.
4. Menú de Chrome > Añadir a pantalla de inicio.
5. Se instala como app.

Nota:
Android/Chrome no permite instalar correctamente una PWA desde archivos locales directos tipo file://. Debe abrirse desde una URL o servidor local.
